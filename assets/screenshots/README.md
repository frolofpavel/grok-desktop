## Tips for great screenshots
- Use a clean project (not this repo itself if it looks too meta).
- Dark theme only (current Grok Desktop identity).
- Hide personal API keys / paths.
- Good window size (1400x900 or wider).
- Show meaningful content: real code, real tool results, real artifacts.
- One focused thing per screenshot.
